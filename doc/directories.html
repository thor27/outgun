Directory structure

This document simply describes the directory structure of Outgun.

Directory namePurpose
client_statsstats in HTML files, saved by the client
cmapsmaps downloaded by the client
configconfiguration files for the server and the client
docdocumentation
fontsfonts, each one in its own file
graphicsgraphic themes, each one in its own subdirectory
languageslanguage files
libLinux libraries
loglog files by the server and the client
mapgenroom templates for the map generator
mappicpictures of the maps are saved here
mapsserver maps, used also by the client
replayreplays
screensscreenshots
server_statsstats in HTML files, saved by the server
soundsound themes, each one in its own subdirectory
srcsource code
